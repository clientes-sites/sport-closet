Sport Closet — Admin de Produtos

Este painel permite editar os produtos do site e gerar o CSV novamente. As imagens podem ser referenciadas manualmente pelo nome do arquivo ou URL nos campos img_1 / img_2 / img_3.

Agora você também pode fazer upload direto para o Cloudflare R2. Informe sua Account ID, Bucket Name e o API Token, selecione o produto e a imagem.

Se quiser usar um domínio público customizado para as imagens, informe a URL no campo R2 Base URL; caso contrário, a URL será gerada com o padrão R2 público.

A senha padrão do painel é sportcloset123. Para alterá-la, edite a constante ADMIN_PASSWORD em admin.js.

Depois de editar, use o botão Exportar CSV para gerar o arquivo atualizado e substituir Tabela_Sport_Closet_2026.csv no projeto.
